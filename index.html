Welcome - Get your HTML refreshed

    First, Let's see few tags :

    Anything placed between these <> braces is known as a Tag.

      Head tag : This tag is mostly used to describe the part of website in browser or to add style.
      Body tag : Here, we can code the content to be displayed on the page of browser.
      P tag : The paragraph tag, used to write lengthy sentences in one go.
      Strong tag : The strong tag, is used to identify an important part in the sentence.
    Strong vs Bold - The difference between the two tags is that the bold tag is intended to draw attention to the text, while the strong tag also highlights the text semantically and indicates that this is an important word or section of text.
      em tag : To get the italics.
      Mark tag: To get the content marked or highlighted.
      del tag: To cross the content.
      ins tag: To get the content, underlined i.e content is inserted
      sub tag: To get the base. X2
      sup tag: To get the power. X2
      abbr tag: To get the abbreviation. PMO

    Comments :

    In html you use comments, to describe a part of code.

    Elements and Attributes

      Element: An element can be a paragraph, image or a video or anything that describes content
      Attribute : An attribute is used to set the properties of an element like width, height, etc. "

    Linking to other page

    We use href attribute to link another page address. Visit Google search Page

    Lists

      Ordered List (ol) - The items are arranged according to order.
      UnOrdered List (ul) - The items are arranged without any specific order.
      Description List (dl) - The items can be described.

    Table

    Creating a table is easy using html

    1. Create a table using table tag.
    2. Use tr tag to create a row
    3. Use td tag to create a column.
    Places India Australia England
    Capitals New Delhi Canberra London

    You can use rowspan and colspan to combine rows and columns

    DIV and SPAN

    This is useful in creating structure of webpages

      DIV - Block element , takes complete width.
      SPAN - Inline element , takes necessary width.

    IFrame
